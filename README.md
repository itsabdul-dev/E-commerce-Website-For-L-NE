# LÜNE — Premium Tea E-Commerce Website

LÜNE is a luxury tea e-commerce web application built with a focus on elegant design 
and a seamless shopping experience. The site showcases a curated collection of premium 
tea bag blends, from herbal and wellness teas to classic and ceremonial varieties, all 
presented through a refined and immersive digital storefront.

## Features

- 🛍️ **Product Catalog** — Browse 16+ premium tea blends displayed in a clean, 
  responsive product grid with real product photography
- 🛒 **Shopping Cart** — Add products to a smooth slide-out cart drawer with quantity
  controls and a live running total
- 💳 **Checkout Page** — A structured checkout flow with contact information and 
  shipping address forms alongside a real-time order summary
- 📦 **Curated Bundles** — Handpicked tea bag collections (Evening Calm & Matcha Energy)
  featured in a dedicated bundles section
- 📬 **Contact Page** — A fully functional contact form alongside business details
- 🎨 **Premium UI** — A "Clinical Luxury" aesthetic with a cream and deep green 
  colour palette, serif typography, and subtle micro-animations throughout

## Tech Stack

- **React** — Component-based UI
- **Vite** — Fast development build tool
- **Framer Motion** — Scroll animations and transitions
- **React Router DOM** — Client-side page routing
- **React Context API** — Global cart state management
- **Vanilla CSS** — Custom styling with CSS variables and Google Fonts
